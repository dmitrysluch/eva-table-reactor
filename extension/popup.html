Manual

  1. Open a data table on any website and press Alt while clicking a cell to capture it.
  2. Give the column a name, then save it to the selected output table.
  3. Repeat on other pages to map additional columns. Each saved column remembers its source URL.
  4. Return here, enter the dates you need, and export the assembled CSV.

Need to adjust mappings? Use the “Manage tables” link to review selectors and URLs.
